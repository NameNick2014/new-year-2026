# New Year 2026 Greeting

Новорічне привітання 2026

## Запуск локально

```bash
cd ./new-year-site
npx serve .
```

Відкрийте: http://localhost:3000

## Розгортання на GitHub Pages

1. Створіть репозиторій на GitHub
2. Завантажте файли з папки `./new-year-site`
3. Увімкніть GitHub Pages в Settings → Pages
4. Виберіть branch: main, folder: / (root)

Сайт буде доступний за адресою:
https://YOUR_USERNAME.github.io/REPO_NAME/
